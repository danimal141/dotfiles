# Markdown Writing Guide

## Formatting Rules

* **リスト記号**: `-` ではなく `*` を使用する
* **水平線**: 見出しの切れ目で `---` を乱用しない（本当に必要な場合のみ使用）

## Examples

Good:
```markdown
# Title

## Section 1

* Item 1
* Item 2
  * Nested item

## Section 2

Content here.
```

Bad:
```markdown
# Title

---

## Section 1

- Item 1
- Item 2

---

## Section 2
```
